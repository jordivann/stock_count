ImAnAccountant.py

Descargar informe de stock desde plex25
Plex 25 > Informes > Stock > Ventas y Stock 

Seleccionar el día a realizar stock y sucursal 28.

El informe darle formato, y guardarlo en .XLXS

Empezar a hacer el stock pistoleando, si no encuentra el código de barra en la base va a pedir una descripción que será de ayuda para los chicos de base de datos para darle una asociación al producto correspondiente. 

Al finalizar descargar el informe final, donde tendremos el resumen y la diferencia de stock entre la teorica y la real para poder hacer el ajuste. 
Tener en  cuenta que si el producto existe con otro código de barra el stock va a tener diferencia si o si, en este caso hay que asociar código.

Se puede reiniciar el conteo, pero reiniciará toda la carga, asi que no pistolear al pedo.
